Relatório de Vendas 3 (Power BI)
- Descrição do Projeto

Este projeto consiste na criação de um relatório interativo de vendas utilizando o Microsoft Power BI, desenvolvido como parte de um desafio prático da plataforma Digital Innovation One (DIO). O relatório foi construído com base em uma base de dados de exemplo, com o objetivo de aplicar conceitos de modelagem de dados, visualização e análise de indicadores de negócio.


- Objetivo

O principal objetivo deste projeto é analisar o desempenho de vendas e transformar dados brutos em informações relevantes, facilitando a tomada de decisões estratégicas. Entre os pontos analisados estão:

Total de vendas

Desempenho por período

Comparação entre categorias ou segmentos

Identificação de tendências e padrões

Análise de indicadores-chave (KPIs)


- Ferramentas e Tecnologias Utilizadas

Power BI Desktop

Modelagem de dados

Criação de medidas com DAX

Visualizações interativas

Base de dados de exemplo fornecida para fins educacionais


- Estrutura do Relatório

O relatório contém os seguintes elementos principais:

Cartões (Cards): Exibição de indicadores principais

Gráficos de colunas e barras: Comparação de vendas por categoria ou período

Segmentações (Slicers): Filtros interativos

Gráficos de linha: Análise de evolução temporal

Painel interativo: Visão geral consolidada das vendas


- Principais Insights Possíveis

Com este relatório é possível:

Identificar períodos com maior volume de vendas

Comparar desempenho entre diferentes categorias

Monitorar tendências ao longo do tempo

Apoiar decisões baseadas em dados


- Como Utilizar

Abrir o arquivo .pbix no Power BI Desktop

Explorar as páginas do relatório

Utilizar os filtros disponíveis para análise interativa

Observar os indicadores e gráficos para extrair insights
